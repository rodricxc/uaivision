# UaiVision
UaiVision: A vision system for IEEE Very Small Size league of robot soccer


## Dependencies
- QT >= 5.0 (tested only on 5.4)
- C++11
- OpenCV (2.4.X - NOT compatible with >=3.0 version yet)
- OpenGL (freeglut3-dev)
